debops.rsyslog
##############



This role manages configuration of ``rsyslog`` server and makes sure that
it's installed.

.. contents:: Table of Contents
   :local:
   :depth: 2
   :backlinks: top

Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v1.7.0``. To install it, run::

    ansible-galaxy install debops.rsyslog
